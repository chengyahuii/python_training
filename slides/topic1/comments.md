# Python 代码注释

Python 中三种代码注释场景

- 单行注释
    ```py {monaco}
    # this code prints "Hello, World!"
    print('Hello, World!')
    ```
- 行内注释
    ```py
    x = 1		# assign numerical value to x
    y = x + 2	# assign the sum of x + 2 to y
    ```
- 多行/块注释
    ```py
    # below code will print 
    # 'Hello, World!' to the screen
    print('Hello, World!')
    ```
    ```py
    """ below code will print 
    'Hello, World!' to the screen """
    print('Hello, World!')
    ```
