# 算术操作符

<br/>
<br/>

| Operator | Meaning | Example |
|---|---|---|
|<kbd>+</kbd>|Addition|x + y|
|<kbd>-</kbd>|Subtraction|x - y|
|<kbd>*</kbd>|Multiplication|x * y|
|<kbd>/</kbd>|Division|x / y|
|<kbd>%</kbd>|Modulus|x % y|
|<kbd>**</kbd>|Exponentiation|x ** y|
|<kbd>//</kbd>|Floor division|x // y|
