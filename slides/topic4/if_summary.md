# if 总结

```py
x, y, z = 7, 4, 2
if x > y and x > z:
    print('x is greater')

# Prints x is greater
```
```py
x, y, z = 7, 4, 9
if x > y or x > z:
    print('x is greater than y or z')

# Prints x is greater than y or z
```

```py
x, y = 7, 5
if not x < y:
    print('x is greater')

# Prints x is greater
```
